# Each App Should Be Able to Do the Following:
- [ ] Handle Basic Todos
- [ ] Preload Todos Prior to Navigation
- [ ] Cache Todos
- [ ] Mutate Todos
- [ ] Handle User Based Todos
- [ ] Display Aggregated Todos Per User
  - [ ] Invalidate Only a Set of Todos for One User, Updating the Collection
- [ ] Display a Loading State if Preloading is Slow (Prior to Navigation)
